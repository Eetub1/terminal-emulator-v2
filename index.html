Supported commands

ls
list directory contents
cd "path"
change directory
echo "arguments"
list own arguments
man "command"
get info about a specific command
mkdir "directory name"
create directory
touch "filename"
create file
rm "filename"
delete file
rmdir "directory name"
delete directory
pwd
print current directory
vim "filename"
open file with text editor

Vim help section

:q
Exit vim without saving
:w
Save changes
i
Go to insert mode to modify file

@
